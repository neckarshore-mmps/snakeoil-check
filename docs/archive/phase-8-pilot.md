# Phase 8 — Pilot + Polish (STUB)

> **STATUS:** Stub — to be filled after Phase 7 ships.

## Goal

Ship to 5 pilot users, gather feedback, polish copy and visuals, and validate the launch-readiness criteria before public marketing.

## Scope

| # | Component | Notes |
|---|-----------|-------|
| 1 | Pilot user recruiting | 5 from German Rauhut's network or LinkedIn outreach. Mix: 2 free-shot-only, 3 paid 3-Shot, 0 power-users yet |
| 2 | Pilot feedback form | After each redeemed shot: 3-question Tally / Typeform — quality, clarity, "would pay again at what price?" |
| 3 | Copy polish | Tone-pass over landing, emails, report disclaimers. German native review |
| 4 | Visual polish | Tailwind / shadcn cleanup, typography pass, mobile breakpoints, hero image (per marketing brief) |
| 5 | Lighthouse pass | Landing 95+ desktop and mobile (CWV: LCP, INP, CLS) |
| 6 | Conversion-funnel tracking | Vercel Analytics custom events: form_view, form_submit, check_complete, upgrade_cta_click, checkout_complete |
| 7 | Marketing prep | LinkedIn-Post v2 from marketing draft, prepared for launch but not posted |
| 8 | Launch decision | After pilot data: green light or one more iteration |

## Definition of Done

- 5 pilot users have completed ≥1 check each
- ≥3 pilot users provided written feedback
- Landing Lighthouse 95+ desktop + mobile, all CWV thresholds green
- All marketing assets prepared (post text, hero image, link landing)
- User says "PASS" — v0.1 launches publicly
- Open items captured for v0.2 backlog

## Dependencies

- Phase 7 Done (legal + hardening complete)
- 5 pilot users identified and committed
- LinkedIn post ready for review
